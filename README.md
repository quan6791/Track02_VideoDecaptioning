# ECCV18_challenge
This is the submission code for ECCV challenges:

the challenge 02: Video decaption


The structure of challenge files:

       --challenge
       
              -- models (please, download models link for track-2 and put here
       
              -- source
               
                      --files
               
                      --predict.bash
       
              -- data_set
       
       
To run the code:
```sh
$ chmod +x predict.bash
$./predict.bash
```
